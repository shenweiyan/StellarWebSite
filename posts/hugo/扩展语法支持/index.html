Columns #

Columns 用于并排显示某些内容（在较窄的屏幕上会自动取消 Columns 效果）。

通过 HTML 注释 <!-- --> 将不同的 Columns 隔开。

        `markdown` text
:::
    !!! note Top Right
        `markdown` text

<!--  -->

:::
    !!! note Bottom Left
        `markdown` text

    !!! note Bottom Right
        `markdown` text

Tabs #

Tabs 用于呈现选项卡效果，允许包含和嵌套任意内容。

当 Tab 没有内容时，对应的选项卡将不可点击。
